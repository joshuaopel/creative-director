# PR #2026 — `feat(brand-studio): hire a Creative Director who builds like a developer`

**Live:** https://joshuaopel.github.io/creative-director/

An interactive job application for **Creative Director, Brand Studio @ GitHub**, built as the thing it's asking for: a pull request.

- **Conversation** — the pitch, a 22-year career contribution graph, and a Copilot review
- **Checks** — the role's required qualifications, all passing
- **Commits** — `git log --career`, from The DAVE School to AAA games to enterprise AI
- **Files changed** — what actually changes when this PR lands
- **The merge button works.** Try it.

## Stack

One hand-tuned `index.html`. No framework, no build step, no dependencies. Vanilla HTML/CSS/JS, GitHub Primer-inspired design system, hosted on GitHub Pages. Designed by Joshua Opel, pair-programmed with Claude Code, shipped with the `gh` CLI — the medium is the message.
